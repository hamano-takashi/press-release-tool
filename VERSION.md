# バージョン管理

## 現在のバージョン

- PDF/Word出力機能
- AI支援機能（ガイド付き作成、自動生成）
- 下書き保存・読み込み機能

## バージョン管理ルール

このプロジェクトは [Semantic Versioning](https://semver.org/lang/ja/) に従います。

- **MAJOR** (1.0.0): 後方互換性のない変更
- **MINOR** (0.1.0): 後方互換性のある機能追加
- **PATCH** (0.1.1): 後方互換性のあるバグ修正

## タグの作成方法

```bash
# 新しいバージョンタグを作成
git tag -a v0.1.0 -m "v0.1.0: リリースメッセージ"

# タグをGitHubにプッシュ
git push origin v0.1.0

# すべてのタグをプッシュ
git push origin --tags
```
